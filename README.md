# NiceChord

Eletric autoharp inspired digital instrument

## Roadmap

### Essentials

  - Chords
    - Three basic variation keys per chord
    - Key combinations variations
  - Harp
    - Chord variations
    - Changes in chords change current playing notes
  
### Extras

  - Drumpads
  - Drum Looper 
  - Chord Looper

## Negative Scope

  - Rhythm
